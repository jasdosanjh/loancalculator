## Loan Calculator

Loan Calculator is a simple web application that allows you to calculate loans with an interest rate.

## Live Demo:
www.jazdosanjh.com/loancalculator/

## Built With
* HTML5
* CSS3
* Bootstrap 4
* JavaScript ES6
